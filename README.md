# Agora v2.1

Agora project with BMAD (Business Model and Architecture Development) methodology integration.

## Overview

This project uses the BMAD Method Module (BMM) for AI-driven agile development, providing comprehensive lifecycle management through specialized agents and workflows.

## Project Structure

- `.bmad/` - BMAD configuration and modules
- `.cursor/` - Cursor IDE rules and configurations
- `docs/` - Project documentation
